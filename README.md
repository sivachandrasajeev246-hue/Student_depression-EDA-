# Student Depression Data Analysis (EDA)

## Description
This project performs exploratory data analysis (EDA) on a student depression dataset to understand how academic, financial, and lifestyle factors relate to depression. The analysis focuses on data cleaning, statistical exploration, and visualization.

## Dataset Overview
- Records: 27,901
- Features include demographics, academic/work pressure, lifestyle habits, and a depression label

## Analysis Performed
- Data cleaning and preprocessing
- Correlation analysis using heatmaps
- Visual analysis using boxplots, count plots, and distributions
- Exploration of factors such as academic pressure, financial stress, sleep duration, and age

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Disclaimer
This project is for educational and analytical purposes only and does not provide medical diagnosis or clinical conclusions.
